# analysis_visualization_system
### 电商评论数据的分析与可视化系统
### 1.项目介绍
本系统基于Hadoop平台，使用Maven进行项目管理，基于ssm框架开发的项目，使用mysql数据库，前端采用JQuery+Bootstrap+ECharts展示数据。
